# MangaHub

A comprehensive manga library management system with CLI, REST API, and multiple communication protocols.

## Features

- Multi-protocol support (HTTP, TCP, UDP, gRPC)
- User authentication and profiles
- Manga library management
- Reading progress tracking
- Real-time notifications
- Community chat
- Data synchronization
- Backup and restore

## Getting Started

See [docs/](docs/) for detailed documentation.

## Project Structure

- `cmd/` - Entry points for different services
- `internal/` - Internal packages
- `pkg/` - Public packages
- `proto/` - gRPC protocol definitions
- `docs/` - Documentation
- `scripts/` - Helper scripts
